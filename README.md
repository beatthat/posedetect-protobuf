posedetect-protobuf
===================

Protobuf types used by posedetect system. Distributed as a pip module for python.

Python Installation
-------------------

```
pip install --user -e git+https://github.com/beatthat/posedetect-protobuf.git@{release-tag}#egg=posedetect_protobuf
```
